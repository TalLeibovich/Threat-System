# SoftwareEngineering_FinalProject

רשימת פונקציות ופונקציונליות:

Citizen
מייצגת אזרח במערכת, כולל נתוניו האישיים כגון רמת האיום, תמיכה בממשלה, השפעה ציבורית, ועוד. מחלקה זו אחראית על חישוב ועדכון רמות האיום של האזרח.
updateThreatLevel: מחשב מחדש את רמת האיום של האזרח על פי פרמטרים אישיים.
setMaxThreatLevel: מעלה את רמת האיום של האזרח למקסימום.
updateThreatLevelToMinimum: מוריד את רמת האיום של האזרח למינימום.
Getters ו-Setters: זמינים לכל השדות הנדרשים במחלקה.

Facility
מייצגת מתקן כליאה. אחראית לניהול הכלואים במתקן, כולל הוספה והסרה של אזרחים, רענון נתוני התפוסה, וניהול דינמי של תפוסת המתקן בהתאם לרמות האיום.
addCitizen: מוסיף אזרח לכליאה במתקן, תוך בדיקה אם ניתן להכניסו או להחליף כלוא עם רמת איום נמוכה יותר.
removeCitizen: מסיר אזרח מהמתקן, מעדכן את סטטוס המעצר שלו, ומנהל מחדש את התפוסה.
refreshFacilityData: רענון נתוני המתקן, כולל חישוב אחוזי תפוסה וסידור הרשימה.
getLowestThreatCitizen: מוצא את האזרח עם רמת האיום הנמוכה ביותר במתקן.
Getters ו-Setters: זמינים לכל השדות הנדרשים במחלקה.

Manager
אחראית על פעולות הניהול במערכת, כולל הוספה ושינוי נתונים של אזרחים, מתקנים, וניהול עדכונים מחזוריים של רמות האיום. המנהל מנהל את נתוני המערכת השוטפים ומתפקד כסמכות ראשונית.
addCitizen: מאפשר למנהל להוסיף אזרח חדש למערכת.
removeCitizen: מסיר אזרח מהמערכת ומעדכן את מערכת הכליאה בהתאם.
addFacility: מוסיף מתקן כליאה חדש למערכת.
printAllCitizens: מציג את רשימת כל האזרחים עם רמת האיום שלהם ומספר ה-ID שלהם.
printAllFacilities: מציג את רשימת כל המתקנים עם קיבולתם, אחוזי התפוסה, ומספרי הזיהוי שלהם.
updateGovernmentSupportLevel: משנה את רמת התמיכה של אזרח בממשלה.
updatePublicImpactScore: משנה את רמת ההשפעה הציבורית של אזרח.
updateEconomicPercentile: משנה את האחוזון הכלכלי של אזרח.
startThreatLevelUpdater: מתחיל תהליך של עדכון מחזורי לרמות האיום של כל האזרחים.

President
מייצגת את הנשיא, שהוא בעל סמכות גבוהה במערכת. הנשיא יכול לבצע חנינות, מעצרי חירום, וייצוא היסטוריית אזרחים. הוא גם אחראי על ניהול רמות איום במקרים מיוחדים.
grantPardon: מעניק חנינה לאזרח, משחרר אותו ממעצר, מוריד את רמת האיום שלו, ומעדכן את מערכת הכליאה.
emergencyDetention: מעלה את רמת האיום של אזרח למקסימום ומכניס אותו למעצר מיידי.
exportCitizenHistory: מייצא את היסטוריית האזרחים לקובץ נתון.
startThreatLevelUpdater: מתחיל תהליך של עדכון מחזורי לרמות האיום של כל האזרחים.
Getters ו-Setters: זמינים לכל השדות הנדרשים במחלקה.

CitizenFacilityAssignment
מתעדת את הקשר בין אזרח למתקן, כולל תאריכי כניסה ויציאה. מחלקה זו מספקת כלים לניהול היסטוריית המעצר של האזרח.
CitizenFacilityAssignment: יוצר קשר בין אזרח למתקן, כולל תאריכי מעצר ושחרור.
setReleaseDate: מגדיר את תאריך השחרור של האזרח.
exportAssignmentData: מייצא את פרטי המעצר לקובץ.
calculateDetentionDuration: מחשב את משך הכליאה בימים.
Getters ו-Setters: זמינים לכל השדות הנדרשים במחלקה.

FacilityThreatAnalysis
מספקת כלים לניתוח נתונים של מתקנים, כולל חישוב ממוצע רמות איום ואחוזי תפוסה. משמשת לאופטימיזציה של ניהול המתקנים.
FacilityThreatAnalysis: קונסטרקטור לניתוח נתוני מתקן כולל אחוזי תפוסה וממוצע רמת איום.
calculateMetrics: מחשב מחדש את הנתונים של אחוזי התפוסה וממוצע רמת האיום.
refreshMetrics: מרענן את נתוני המתקן לפי שינויים עדכניים.
getAverageThreatLevel: מחזיר את ממוצע רמת האיום במתקן.
getOccupancyRate: מחזיר את אחוזי התפוסה במתקן.
calculateOverallOccupancyRate: מחשב את אחוזי התפוסה הכוללים בכל המתקנים במערכת.
getFacilityIdWithLowestOccupancyRate: מחזיר את מזהה המתקן עם אחוזי התפוסה הנמוכים ביותר.
getFacilitiesAboveOccupancyRate: מחזיר רשימה של מתקנים עם אחוזי תפוסה מעל סף מסוים.
Getters ו-Setters: זמינים לכל השדות הנדרשים במחלקה.

ThreatManagementSystem
המחלקה המרכזית שמנהלת את כל תהליכי המערכת: מעקב אחר אזרחים, מתקנים, ניהול מערכת הכליאה, ומימוש אלגוריתמים לניהול רמות איום ותפוסת מתקנים.
addCitizen: מוסיף אזרח למערכת ומעדכן את מערכת הכליאה.
removeCitizen: מסיר אזרח ממערכת הכליאה, מעדכן את הסטטוס שלו, ומנהל מחדש את מערכת הכליאה.
addFacility: מוסיף מתקן חדש למערכת ומנהל מחדש את כל מערכת הכליאה.
removeFacility: מסיר מתקן מהמערכת, משחרר את כל הכלואים שבו, ומנהל מחדש את מערכת הכליאה.
*manageDetention*: מנהל את תהליך ההקצאה והסרה של אזרחים במתקני הכליאה בהתבסס על רמות איום, רף סטטי, ורף דינמי.הפונקציה החשובה ביותר שמממשת את האלגוריתם .
getStaticThreshold: מחזיר את רמת האיום הסטטית שנקבעה מראש.
getDynamicThresholdForAllFacilities: מחזיר את רמת האיום הנמוכה ביותר מבין כל הכלואים במערכת.
getFacilityWithLowestOccupancy: מוצא את המתקן עם אחוז התפוסה הנמוך ביותר.
refreshThreatLevels: מעדכן את רמות האיום של כל האזרחים במערכת.
Getters ו-Setters: זמינים לכל השדות הנדרשים במחלקה.

Main
main: מנהל את תהליך העבודה הכולל של המערכת. מציג תפריט ראשי, מאפשר גישה לנשיא או למנהל, צפייה בפרטי מתקנים, וסיום עבודה.
handleManagerLogin: מטפל בתהליך התחברות של מנהל, כולל בדיקת אישורי כניסה.
handlePresidentLogin: מטפל בתהליך התחברות של נשיא, כולל בדיקת אישורי כניסה ומפתח סודי.
managerMenu: תפריט פעולות למנהל, כולל ניהול אזרחים ומתקנים, עדכוני נתונים, והתחלת תהליכים.
presidentMenu: תפריט פעולות לנשיא, כולל מתן חנינה, מעצר חירום, וייצוא היסטוריית אזרחים.
addNewCitizen: מאפשר הוספת אזרח חדש למערכת עם פרטי קלט מלאים.
updateCitizenSupportLevel: מעדכן את רמת התמיכה של אזרח בממשלה.
updateCitizenPublicImpactScore: מעדכן את רמת ההשפעה הציבורית של אזרח.
updateCitizenEconomicPercentile: מעדכן את האחוזון הכלכלי של אזרח.
viewFacilityDetails: מציג את כל האזרחים הכלואים במתקן מסוים.
exportHistory: מייצא את ההיסטוריה של האזרחים לקובץ נתון.
emergencyDetentionForCitizen: מאפשר לנשיא לבצע מעצר חירום של אזרח לפי ID.
יש עוד מלא דברים שאפשר לעשות בתפריט אלה רק חלק .
